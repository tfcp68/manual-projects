Самое большое количество нулей на конце при длине n будет всегда равно 1. Кроме случая с одним нулём на конце, 
ячейка (i)(j) будет равна (i-1)(j-1). Таким образом заполняем всю таблицу.

1. Подзадачей служит поиск поиск кол-ва чисел длины i c j-ым количеством нулей на конце 

2. Рекуррентные формулы: 
	1. Если i =0: $a_{[i][j]} = 2^{j - 1}$
	2. $a_{[i][j]} = a_{[i - 1][j - 1]}$

3. Начальное значение: $a_{[0][0]} = 1$

4. Необходимые значения хранятся в матрице $a[n][n]$

Ответом будет общее количество всех возможных чисел с заданной длиной минус сумма всех ячеек в полученной матрице $a[i][j]$, где $3<=i<n$, $3<=j<n$.
```
Псевдокод:

если j = 0 и i = 0, то
	a[i][j] <- 1
иначе, если i = 0, то
	a[i][j] <- 2 в степени (j - 1)
иначе
	a[i][j] <- a[i - 1][j - 1]
для i от 3 до n
	для j от 3 до n
		сумма <- сумма + a[i][j]
ответ <- 2 в степени n - сумма
```
```python
#python

def petshop_matrix(n):
    if n < 0:
        return -1
    matrix = [[0 for i in range(n)] for j in range(n)]
    for i in range(n):
        for j in range(i, n):
            if j == 0 and i == 0:
                matrix[i][j] = 1
            elif i == 0:
                matrix[i][j] = 2 ** (j - 1)
            else:
                matrix[i][j] = matrix[i - 1][j - 1]
    sum_3 = 0
    for i in range(2, n):
        for j in range(i, n):
            sum_3 += matrix[i][j]
    answer = 2**n - sum_3
    if n == 0:
        answer = 0
    return answer
```

1. Подзадачей будет поиск поиск количества бин. последовательностей i-ой длины с тремя и более нулями подряд.

2. Рекуррентная формула: $a_{n} = a_{n-1} + 2^{n-3}$. 

3. Стартовое значение

4. Значения будут записываться в массив $a[n+1]$

В задаче необходимо найти количество последовательностей с обратным свойством, поэтому ответ будет высчитываться так: $count = 2^{n} - a_{n}$

```
Псевдокод:
//n - длина бинарной последовательности
//a - массив, где в i-ой ячейке количество чисел длинной i с тремя нулями подряд

если n < 0, то
	вернуть -1
если n = 0, то
	вернуть 0
иначе, если n < 3, то
	вернуть 2^n
для i от 3 до n
	a[i] <- a[i - 1] + 2^(i-3)
ответ <- 2 в степени n - a[n]
```
```python
#python

def petshop_array(n):
    if n < 0:
        return -1
    if n == 0:
        return 0
    if n < 3:
        return 2**(n)
    a = [0] * (n + 3)
    for i in range(3, n + 1):
        a[i] = a[i - 1] + 2 ** (i - 3)
    answer = 2 ** n - a[n]
    return answer
```
